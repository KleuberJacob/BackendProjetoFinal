# Back-end E-Commerce XMarket
Este é o back-end de um site localhost que simula uma empresa de revenda de tênis Nike chamada XMarket. \
O site é referente ao projeto final do programa Araujo Tech, uma parceria entre a Drogaria Araujo e a Gama Academy que visa contratar e treinar novos desenvolvedores back-end em Java. \
Repositório front-end do site: https://github.com/KleuberJacob/BackendProjetoFinal
